# 🦆 DuckShot

DuckShot is a community-driven 2D platformer shooter built in Unity. Players use a pixel-art shotgun to clear handcrafted levels filled with ducks across platform-based environments. The project is open source, beginner-friendly, and designed to grow with community feedback while maintaining a professional structure.

---

## 🎮 Core Features

* Pixel-art shotgun combat
* Pixel-art duck enemies
* 2D platformer gameplay
* 10 base levels (initial goal)
* Multiple duck behaviors
* Community-driven updates
* Clean and readable codebase

---

## 🎨 Art Style

DuckShot uses simple pixel art for clarity and accessibility:

* Pixel-art shotgun
* Pixel-art duck enemies
* Minimalist environments
* Consistent pixel scaling
* Beginner-friendly asset pipeline

The goal is to keep visuals clean while allowing contributors to easily add new content.

---

## 🚀 Project Goals

* Create 10 polished base levels
* Implement multiple duck types (walker, jumper, charger)
* Add shotgun mechanics (spread, recoil, polish)
* Maintain clean and modular code
* Expand with community suggestions
* Build a collaborative open-source game
